=====================================
弾幕ゲーム - DANMAKU GAME
=====================================

【ゲーム概要】
5体のボスを撃破していく本格的な弾幕シューティングゲームです。
美しい弾幕パターンを避けながら、敵を倒してハイスコアを目指しましょう！

【メニュー画面】
・ゲームスタート: 難易度選択画面へ
・操作説明: 操作方法の確認
・ハイスコア: 上位3つのスコアを表示

【難易度選択】
・EASY: 弾速70%、初期ライフ5、初期ボム5
・NORMAL: 通常の弾速、初期ライフ3、初期ボム3
・HARD: 弾速150%、初期ライフ2、初期ボム2

【操作方法】
・矢印キー: 自機の移動
・Shift: 低速移動（精密な避けに使用）
・Z: ショット
・X: ボム（画面上の弾を消去）
・ESC: ポーズ
・M: ポーズ中にメニューに戻る

【ゲームシステム】

■ライフシステム
・初期ライフ: 3
・被弾するとライフが減少
・ライフが0になるとゲームオーバー

■パワーシステム
・敵を倒すとパワーアイテムが出現
・パワーが上がるとショットが強化
・最大パワー: 3.0

■ボムシステム
・初期ボム数: 3
・使用すると画面上の弾を消去
・被弾後は3個に回復

■スコアシステム
・通常敵撃破: 100点×現在のステージ数
・ボス撃破: 1000点×ボス番号
・グレイズ（弾をかすめる）: 10点

【ステージ構成】

■通常敵（ステージごとに変化）
ステージ1: 菱形・オレンジ色 - 単純な円形弾
ステージ2: 円形・水色 - 円形弾＋自機狙い弾
ステージ3: 三角形・緑色 - スパイラル弾
ステージ4: 五角形・ピンク色 - 高速移動＋複合弾
ステージ5: 星形・黄色 - 全方位弾＋スパイラル

■ボス一覧
1. 赤魔王（HP: 100）
   - 六角形の赤いボス
   - 花弾幕と円形弾を展開

2. 青竜王（HP: 150）
   - 八角形の青いボス
   - 波状攻撃とレーザー攻撃

3. 緑妖精（HP: 200）
   - 円形＋衛星の緑のボス
   - スパイラル弾とランダムバースト

4. 紫天使（HP: 250）
   - 四角形の紫のボス
   - 十字弾幕とホーミング弾

5. 金光神（HP: 300）
   - 星形の金色のボス
   - 黄金螺旋とメガバースト攻撃

【攻略のコツ】
・Shiftキーで低速移動すると、自機の当たり判定（赤い円）が表示されます
・ボムは緊急回避に使いましょう
・パワーアイテムは優先的に取得してショットを強化
・弾をギリギリでかわすとグレイズボーナスがもらえます
・各ボスの攻撃パターンを覚えることが重要です

【アイテム】
・Pアイテム（紫）: パワーアップ
・$アイテム（緑）: スコアボーナス

【ゲームクリア条件】
5体のボスをすべて撃破するとゲームクリア！
最高スコアを目指して何度でも挑戦してください。
